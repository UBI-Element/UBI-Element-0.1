## 第一部分：UBI的历史趋势

自无条件基本收入（UBI）被阐释以来，众多定义被赋予在以“国家”为背景的框架下解读并实践。在业内，最经典的案例即是让AI给大家发钱的事，这是将人类顶尖技术产生的经济成果分发给“实践”用户的经典理念。所谓实践用户可以理解为web3er在某特定场景中予以实践的人。

我们计算一组直观的数据，全球人数80亿，年总GDP100万亿美金，假设全部用于消费，税收总率10%，“货币乘数”倍率我们暂定为4-1=3（经济学中同一块钱在流通中可以产生的总支出数量，即多次使用的次数），可以得出年人均为1.25万美金，那么在不考虑金融环境的前提下，一年之中1.25万美金的纯消费产生的价值总额为1.25万*3*80亿=300万亿（三倍于年GDP），可见，仅仅需要为每个人提供4千多美金即可达成目前的全球GDP目标。按10%总税率，年总GDP不变，不出20年就能解放全人类的200万亿负债。

当然以上计算纯属示意，不代表准确度量，我们参照这种逻辑，很明显的可以感受到一个事实，那就是价值的产生似乎是凭空而来。果真如此吗？我们不妨据实理论，当每人分发1.25万美金的时候，显然它是满足绝大数人们的日常生活所需，理想状态下人们在吃喝不愁的前提下会去发扬自己的兴趣爱好等，以实现兴趣驱动的社会发展驱动力。不理想状态下就是人们陷入了舒适区而没有实质价值的产生，导致1.25万美金迅速贬值，购买力下降，最终导致GDP下跌，均分额度不足以满足生活所需，这时候的均分额度可能又悄悄的变回为4000美金。

假设分发额度为4000美金，既满足了人们的基本生活保障，又实现了消费型激励，同时构建了依托UBI理念的经济体系，当人的群体不为衣食住行的基本需求而劳作时，社会的文明形态将发生质的改变。当然这也是据实理论下的理想状态。我们需要借鉴的是这种构思逻辑或者形式。并不是说一直参照固有的经济范式去思考UBI，这一切都需要我们的不断探索，当然我们自然可以通过上述的这种构思形式远观的看到其价值和意义的所在，这也正是我们动力的来源。

人类从未有过现在的显性的互联集体意识，这归功于信息时代通讯技术的发展，当我们不完全跳出UBI（无条件基本收入）这个逻辑下进行场景叙事时，我们遇到最大的问题和困难是价值如何产生？钱从哪来？这是两个“致命”问题。尽管所有事业都存在这样的命题，但在UBI中却尤为明显。我们先刨除这两个问题，假定它没有问题或者已经解决，那么在接下来我们站在（UBI）系统独立的角度去追溯我们要做的事情，看最终是否能够明晰问题的答案（相关论述在公论中已有说明）。

首先假定我们的有效用户基数足够，再假设我们的系统结构完善或者产品应用展示清晰。此时作为一个独立系统或者产品其最大的问题就和普遍的产品一样，存在被消费的可能，因为其能为别人带来经济实惠、高效、便宜等结果，所以才产生了价值。那么UBI作为独立系统或者产品能为别人带来什么样的价值？

1、普世的web3价值观，话事人认为web3的根本意义就是“价值重分配”，它天然具备价值分配结构的重新定义的能力，同时这也是区块链产生时先辈们的终极心愿之一，而UBI却恰恰能够承担起这样的命题。检验一个社区或DAO组织是否足够的趋近于理念上的“匀称”，其能否采纳UBI是一个重要的参考依据。这种“匀称”是治理层是否具备web3价值观的有效见证。

2、相对的普遍适用，在DAO社区中，很多时候，我们都在强调贡献标记，同时尽可能的公平及均衡分配，我们叫他POW，本质上说完全标记下，这种形式类似于UBI的逻辑，因为完全标记不仅仅标记了结果，也标记了过程，但我们常常以结果导向，而很多时候，结果导向是不能适用的，比如我们在起初的时候，在一定范围内可能热度高昂，兴趣浓厚，但起初之时，DAO组织或许仅仅以兴趣发起，并没有明确的发展方向，更没有可用产品应用，那么此时更多的贡献并没有结果，而我们却将这种贡献度归纳为早期贡献者，以在时空机遇中给予更多的价值标记，但这并不完全可靠，一方面的原因是大家并没有发现一个良好的方式去解决在早期时贡献者的价值标记问题，另一方面是我们总认为前期的价值一定是高的，同时也认为后期的贡献价值在同条件下一定是客观降低，同时也认为，前期贡献者的获利一定要高于后来者，这是一种自然偏见或者思维偏差。但现在我们可以借用UBI的理念去行事，以在最早期进行最小贡献标记。这或许是一个挺耗费工作量的事情，尽管有人可能认为“不值当”，但话事人认为，想要在DAO组织的基本经济结构上改变未来的话，这一点必不可少，而且会行之有效。

3、外在性，如果你在阅读以上文字时站在社区内的视角去审视了这个事情，那么请你跳出这个视角来重新审视，它有两个好处，一、常言道他山之石可以攻击玉，这是借助了排他性的优势一面：二、省钱，我们前面提到过致命问题-钱从哪来，如果从内部来，一定是需要审视笔钱的来处与用法，其中的过程性困难无疑是阻碍UBI落地的最大障碍。如果这个钱是从外部而来，那么这个问题或许能直接迎刃而解。一般情况下仅仅需要成员之间达成UBI可用的共识。

## 第二部分：“组织”建设与产品应用的可能性

我们从两个范围进行说明，一个是广义上大家普遍接受的UBI定义，一个是狭义的有限范围的UBI。这两者区别大概是前者建立在以国家为范畴的角度，后者仅仅是一个组织一个群体。

我们认为UBI组织仅仅是一个、人的群体，它有别于DAO组织，公司或者项目构成，但在我们话事、策展、构建的同时，却一定同属于这三者。

广义上我们每个参与者都将以超级个体的身份参与到UBI的构建中，或者话事中，或者策展中，以在UBI生态中获得自我价值。

狭义上我们将组织一个群体，通过UBI的第三者身份之标记（最小贡献标记）等功能的实现以赋能于其他社群组织、DAO生态、项目产品中，等。

有关广义及有限范围UBI在价值观上的区分，请参照《UBI公论》部分。

当我们组建社群后，我们首先应该考虑的是通过什么样的事物将大家维系在一起，首先我们将定义一个最小闭环，以实现参与者极简的参与其中。我们先设想一下这些最小闭环的特征，然后在后文具体展开。据话事人在话事过程中的经验判断，有明显互利或依存关系的事项中其UBI实践的适配性越高，如：两方关系中的XXX（见案例文或下文将有具体展开），三方关系中的（垄断）平台与消费者之间。

### 第一：社群的组建

我们以社区的名义去筹备它，在社区里，更具备一种垂直和或许高效的统一动效，这正是社区与DAO之间的区别吧。但治理流程上我们完全遵循DAO的逻辑，采取自愿自主自驱动原则。

我们前面假设了有效用户基数足够的情况，但它并不能一蹴而就，我们需要从零开始，我们为参与者设定了简单的三个身份组，一、话事人、二、标记员、三、策展人。

一、话事人：有着由内向外进行活动的行为表征，更多的是将UBI的观念传递给广泛的人群和社群以便其借用或使用我们的UBI策略。当然这个策略是“第三方相关”的，正如上文所说的“第三者身份之标记”一样，对于DAO组织来说，这是一个外在的第三方。有关话事人具体的事务权益及行为准则，我们后文再述。

二、标记员：在UBI策略开展阶段，提供行为、信息或者贡献标记的人。更多时候社区成员大都是标记员身份，并且同时也是其他DAO组织的成员。这点毫无冲突，标记员仅在达成UBI策略合作后，在双方许可前提下进行了标记工作。这是一种互利。

三、策展人：我们允许策展人或标记员或话事人等进行UBI的策展，策展方式不限于业务宣传，板块拓展，商务代理，设计，开发，运营等。但任何基于UBI-Element名义或内容来处的须载明出处。

身份的获得过程代表着有限范围组织或者UBI.Element在根据自己规则在挑选自己的成员或者“国民”

UBI-Element社区的权益规则将遵从最大分配原则，即除去必要留存和必要发展支出等，剩余部分全额发放。同时在分配之前将周期性归整标记员贡献。当然这些都是后话，在此之前我们需要先创造价值。

### 商业化规划

有效行为是产生有价价值的基础。

作为普通用户参与到UBI生态中，这是一个基本路径，同时他也是UBI生态基本价值的创造者，可以获得绝大部分的UBI准则下的分配。

每个人都有属于自己能力范围内的收益，它可以是我们生活的基本保障，每个项目产品、系统组织，国家机构，都有这样潜在的愿望，它希望在这个系统中的生产者都有一份基本保障，当然这里指的并不完全是无条件基本保障。因为我们在思考这类问题时需要不断地去区分基本收益和基本保障在系统中的不同效应，尽管我们已经很简约的去实现它。在UBI-Element中的三组角色里，它们有着不同的身份特征，每个身份代表了不同的个人能动，同时在不同能动的前提下，会产生不同系统性效益，这是我们区分三者权益的准则之一。

标记员将是这个系统中最大的贡献者和受益者。

因为考虑商业化进程，我们区分了身份组对应的不同分工和不同权益。话事人作为重要的传播窗口，我们将其与商业化资本关联起来。在前期资本作为标记员贡献者的信心支撑，在标记员贡献价值未能产生集体价值或者收益时，商业资本对于标记员身份来说是一种激励。当标记员贡献产生了集体价值或者盈利之后，这种价值将反馈与资本，但这种反馈是有限的，这和一般性商业投资收益是正相关的，而且在一定时效内，如一年期，标记员身份组贡献并未能产生实质性价值时，资本将可选择性自愿退出，此时退出的负效益和投资正收益可能两相对等。

所以这里看出作为话事人的权益大多集中在资本运营测，而作为策展人可能有一部分也是重叠这些权益，但重点在于策展行为是不受EBI-Elemen约束的，它有着随同UBI-Element允许规则下的自由策展权。

所以UBI-Element期望在众多标记员的参与下将它打磨成UBI-ECO，让UBI生态成为我们理想中的那个WEB3世界。

### 产品路线动态策划

如果内部需要一项技术开发，也一定是遵循了有人自愿承担的自愿行为。否则，我们将持续维系之前的功能展示需要。如果再需要，我们就筹集捐赠自己的UBI奖励或者配额，或者，通过话事人和策展人借以资本的响应。

治理规则

我们采用重统筹轻治理的社区治理方式，避免DAO中的分散化信息传递与专断性可能。

身份组中，话事人起到向外拓展的作用，在社区建设中起到“推波助澜”的作用，故有治理责权，主要负责提案以在业务合作中确定合作案；标记员是社区的生力军，主要负责投票确认，同时有提议权，投票规则另记，大概以半数通过后确定合作意向，修正后三分之二数通过可为执行（不排除在修正可能）。策展人可以是非社区成员，可以根据UBI-Element规则自由策展的人或组织。当然对于策展人UBI-Element对其并不会进行标记或者激励，同时在治理上策展人同样没有相应权责。

捐赠：话事人在向外构述UBI-Element规则时，有个心理预期，即希望获得对方社区在认同UBI的同时可以向UBI-Element捐赠一定数量的Token（不限币种）。捐赠规则另附。话事人手册也将由话事人共同合力修订。

公益捐赠：前面我们提到了实现特定行业的UBI适用，如公益项目，在UBI中，话事人认为以UBI的形式实现公益是一件非常有意义的工作，如果我是一个公益工作者，那么我一定会这样去做，但尽管目前不是，但我期望有那么一天，UBI-Element有能力开启这样的公益事业，而在UBI-ECO中，它将被设定为目标之一。
